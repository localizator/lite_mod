    Установка

    1) Скачиваем прошивку и копируем ее в корень карты памяти (/sdcard)
    2) Заходим в Recovery:
    На выключенном телефоне зажать кнопки (Громкость вверх+включение) 3) Выбираем Install zip from sdcard, далее Choose zip from sdcard
    4) Выбираем прошивку и подтвеждаем установку
    5) Ждем надписи install from sdcard complete
    6) Выбираем Reboot system now
    7) Ждем пока загрузится телефон
    (Навигация по Recovery осуществляется при помощи кнопок громкости (вверх/вниз), включения (выбор), сенсорной кнопки назад)
